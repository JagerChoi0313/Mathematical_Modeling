## 4 符号说明

为便于建立露天矿车辆安排模型，本文将铲位、卸点、车辆、运输趟数和产量等信息统一符号化。主要符号如下。

### 4.1 集合与下标

| 符号 | 含义         | 说明                         |
| ---- | ------------ | ---------------------------- |
| i    | 铲位编号     | i = 1, 2, …, 10              |
| j    | 卸点编号     | j = 1, 2, …, 5               |
| k    | 卡车编号     | k = 1, 2, …, 20              |
| I    | 铲位集合     | I = {1, 2, …, 10}            |
| J    | 卸点集合     | J = {1, 2, …, 5}             |
| J₁   | 矿石卸点集合 | 包括矿石漏、倒装场Ⅰ、倒装场Ⅱ |
| J₂   | 岩石卸点集合 | 包括岩石漏、岩场             |
| K    | 卡车集合     | K = {1, 2, …, 20}            |

其中，本文将卸点编号规定如下：

| 卸点编号 | 卸点名称 | 卸点类型 |
| -------- | -------- | -------- |
| 1        | 矿石漏   | 矿石卸点 |
| 2        | 倒装场Ⅰ  | 矿石卸点 |
| 3        | 倒装场Ⅱ  | 矿石卸点 |
| 4        | 岩石漏   | 岩石卸点 |
| 5        | 岩场     | 岩石卸点 |

因此有：

J₁ = {1, 2, 3}

J₂ = {4, 5}

------

### 4.2 已知参数

| 符号  | 含义                                          | 单位    |
| ----- | --------------------------------------------- | ------- |
| d_ij  | 铲位 i 到卸点 j 的距离                        | km      |
| O_i   | 铲位 i 的矿石储量                             | 万吨    |
| R_i   | 铲位 i 的岩石储量                             | 万吨    |
| g_i   | 铲位 i 矿石的平均铁含量                       | %       |
| D_j   | 卸点 j 的最低产量要求                         | 万吨    |
| L     | 卡车单次满载运输量                            | 万吨/车 |
| T     | 一个班次的工作时间                            | min     |
| t₁    | 电铲平均装车时间                              | min/车  |
| t₂    | 卸点平均卸车时间                              | min/车  |
| v     | 卡车平均速度                                  | km/h    |
| C_ij  | 卡车在路线 i → j 上完成一次往返运输的循环时间 | min     |
| H_ij  | 一辆卡车在路线 i → j 上一个班次最多运输趟数   | 趟      |
| A     | 一台电铲一个班次最多装车次数                  | 趟      |
| B     | 一个卸点一个班次最多卸车次数                  | 趟      |
| G_min | 矿石卸点允许的最低平均品位                    | %       |
| G_max | 矿石卸点允许的最高平均品位                    | %       |

根据题目条件，有：

L = 154 吨 = 0.0154 万吨

T = 8 × 60 = 480 min

t₁ = 5 min/车

t₂ = 3 min/车

v = 28 km/h

G_min = 28.5%

G_max = 30.5%

一台电铲一个班次最多装车次数为：

A = 480 ÷ 5 = 96 趟

一个卸点一个班次最多卸车次数为：

B = 480 ÷ 3 = 160 趟

卡车在路线 i → j 上完成一次运输循环，包括装车、去程运输、卸车和返程运输，因此循环时间为：

C_ij = 5 + 3 + 2 × d_ij ÷ 28 × 60

即：

C_ij = 8 + 120d_ij ÷ 28

一辆车在路线 i → j 上一个班次最多运输趟数为：

H_ij = 向下取整(480 ÷ C_ij)

### 4.3 决策变量

| 符号  | 含义                                   | 变量类型     |
| ----- | -------------------------------------- | ------------ |
| x_ij  | 一个班次内从铲位 i 到卸点 j 的运输趟数 | 非负整数变量 |
| y_i   | 铲位 i 是否布置电铲                    | 0-1 变量     |
| n_ij  | 路线 i → j 上安排的卡车数量            | 非负整数变量 |
| z_ijk | 第 k 辆卡车是否安排在路线 i → j 上     | 0-1 变量     |

其中：

当 y_i = 1 时，表示铲位 i 布置电铲；

当 y_i = 0 时，表示铲位 i 不布置电铲。

当 z_ijk = 1 时，表示第 k 辆卡车被安排在铲位 i 到卸点 j 的路线；

当 z_ijk = 0 时，表示第 k 辆卡车不在该路线运输。

路线 i → j 上安排的车辆数可由 z_ijk 汇总得到：

n_ij = z_ij1 + z_ij2 + … + z_ij20

也就是说，n_ij 表示该路线一共安排了几辆车。

------

### 4.4 产量与运量表达式

由于每辆卡车每次满载运输 L = 0.0154 万吨，因此路线 i → j 的运输量为：

Q_ij = L × x_ij

整个班次的矿石产量为：

所有运往矿石卸点的运输量之和。

即：

矿石产量 = L × 所有矿石路线运输趟数之和

整个班次的岩石产量为：

所有运往岩石卸点的运输量之和。

即：

岩石产量 = L × 所有岩石路线运输趟数之和

路线 i → j 的单趟运量为 154 吨，运输距离为 d_ij km，因此该路线的总运量为：

154 × d_ij × x_ij

整个班次的总运量为所有运输路线吨公里之和。
